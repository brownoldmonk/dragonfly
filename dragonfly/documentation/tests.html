Test script¶

Main test runner script¶

This file is Dragonfly’s main test scripts. Running it will execute all tests within the Dragonfly library.

Dependencies¶

This script uses several external libraries to collect, run, and analyze the Dragonfly tests. It requires the following external dependencies:

  • Nose – an extension on unittest.
  • Coverage – a code coverage measuring tool.
run_nose()¶
Use Nose to collect and execute test cases within Dragonfly.
run_pep8()¶
Use the pep8 package to analyze the Dragonfly source code.
run_pylint()¶
Use the pylint package to analyze the Dragonfly source code.
